Javascript Variables

Definition

A variable or var in javascript is a "container" that can be set to a value in a variety of ways. It can then be referenced or changed by other things.

Code example

var favoriteNumber = 16;
var greeting = "hello"

creates a variable called favoriteNumber and sets it equal to the number 16, and creates a variable called greeting and sets it equal to the string "hello".
