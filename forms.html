Forms

Definition

This allows a visitor to your website to submit information athat Javascript can interpret.

Code example

<div class="container"> <h1>Submit some information</h1> <div class="container" id="blanks"> <form> <div class="form-group"> <label for="yourName">What is your Name? </label> <input id="yourName" class="form-control" type="text" placeholder="Write a name here"> </div> <div class="form-group"> <label for="yourQuest">What is your Quest?</label> <input id="yourQuest" class="form-control" type="text" placeholder="Write some words here"> </div> <div class="form-group"> <label for="favColor">What is your favorite Color?</label> <input id="favColor" class="form-control" type="color"> </div> <button type="submit" class="btn">Submit!!!</button> </form> </div> $("#blanks form").submit(function(event){ var nameInput = $("input#yourName").val(); var questInput = $("input#yourQuest").val(); var colorInput = $("input#favColor").val(); $(".yourName").text(nameInput); $(".yourQuest").text(questInput); $("#colorized").css('color', colorInput); $("#story").show(); event.preventDefault(); });
