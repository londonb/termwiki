Divs and Spans

Definition

A "Div" tag is used to hold a group of elements (so you could adjust ids and classes, etc).

A "span" tag is used inline to affect something within an element to add id or class or something.

Code example

<div class="happy"> <h3>Everything in this div is happy</h3> <p> Except, this span makes this <span id="sad">word</span> be "sad"</p> </div>

See it in Action?

Everything in this div is happy

Except, this span makes this word be "sad"

"happy" and "sad" are set via .css .
