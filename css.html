CSS

Definition

CSS (Cascading Style Sheets) are that are used to apply styles across multiple pages of a website. They allow you to make changes to style in one place that affect multiple pages.

Code example

This code written in our .css file makes anything with the class "happy" have yellow text with a black background or anything with an id of "sad" have blue text.

.happy{ color: yellow; background-color: black; } #sad { color: blue; }

See it in Action?

Everything in this div is happy

Except, this span makes this word be "sad"
