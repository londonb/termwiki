Javascript Methods

Definition

A wise website once wrote: "You can think of a method as a message that you send to a number or variable, and the result that JavaScript gives you as the number's response to that message."" They can have parameters (variables that take an argument) in parentheses.

Code example

This would set a variable as the string "hey" and perform a method that would convert all of the characters in the string to uppercase.

var word = "hey";
word.toUppercase();
